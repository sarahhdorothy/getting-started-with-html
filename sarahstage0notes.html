Unit 1: Getting Started

How the Web Works

The Web is composed of three important elements that all have to work together in order for the web to work. First we have a person sitting on their computer using it to search things in a browser.When they try to go to any certain website,the internet has to send http requests to a server. The server then finds the websites code, called html, to send back to the computer. In the middle of all of that, the server might have to find links and references to images, videos, and other documents.


HTML

HTML stands for HyperText Markup Language and it is the language that programmers use
to write their code and websites.


How Does Coding Work?

Coding is the tool that all programmers use to create the pages that everyone sees on the internet. In coding, there are different pieces that have to be used at specific times. There are tags and elements, inlines and blocks, images and links. It is difficult to keep them all straight and to use them in the correct way.


Programming is AWESOME!

Programming is a skill that will be used for many years to come. But, it can also be a frustrating profession to choose. There are so many tags, elements, attributes, and changes that are put into HTML, that sometimes it is hard to remember all of them. If there is ever a time when one is stuck on a markup or CSS class, look it up! The internet is there to help!


Computers are Stupid

Computers are not necessarily stupid. But they do take everything literally. It is imperative that you double check your code and make sure that every comma and closing tag is in place. Otherwise, the computer will not understand it!
